# ai_interview_system

InterviewAI Platform is a single-page HTML application — an AI-powered mock interview tool with a dark cyberpunk aesthetic (deep navy/black background, cyan and amber accents).
Here's what it includes:

Auth screen — Login/signup interface to get started
Setup screen — Configure your interview (role, difficulty, question type)
Interview screen — Live mock interview session with AI-generated questions
Results screen — Performance dashboard with charts (Chart.js) showing scores, feedback, and analytics
Design — Space Mono + Outfit fonts, glowing cyan UI, grid overlay background, smooth screen transitions
